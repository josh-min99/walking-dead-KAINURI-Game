# Walking Dead - Unity Multiplayer Board Game

군인 vs 좀비의 전략적 턴제 보드게임 프로젝트

## 프로젝트 개요

Walking Dead는 Unity로 개발된 1대1 멀티플레이어 턴제 전략 보드게임입니다. 플레이어는 군인 또는 좀비 진영을 선택하여 상대와 대결하며, 전략적인 유닛 배치와 아이템 활용으로 승리를 목표로 합니다.

## 주요 특징

### 게임플레이
- **턴제 전략 게임**: 10턴 시간 제한 내 전략적 이동 및 행동
- **비대칭 진영**: 군인과 좀비, 각기 다른 특성과 전략
- **147개 타일 보드**: 대규모 전략적 공간
- **5단계 유닛 시스템**: 각 진영별 5단계 업그레이드 가능

### 게임 시스템
- **상점 시스템**: 골드를 사용한 유닛 및 아이템 구매
- **다양한 전략 아이템**:
  - 방탄복, 구급상자, 백신
  - GPS 추적기, 지뢰, 메가폭탄
  - 인과역전, 텔레포트 등
- **실시간 전투 시스템**: 유닛 간 전투 및 승패 판정
- **미니게임**: 추가 보상 획득 기회

### 기술 스택

**게임 엔진**
- Unity 2D/3D

**네트워킹**
- FishNet - 고성능 멀티플레이어 네트워킹 라이브러리
- TCP 기반 서버-클라이언트 구조
- Host/Connect 방식의 P2P 연결

**개발 언어**
- C#

**UI/UX**
- TextMeshPro
- Unity Canvas System
- 반응형 UI (AspectRatioFitter)

## 게임 구조

### 진영별 유닛
**군인 진영**
1. 신병
2. 보병
3. 특전사
4. 사이보그
5. 슈퍼솔저

**좀비 진영**
1. 아기좀비
2. 일반좀비
3. 근육좀비
4. 특수좀비
5. 좀비킹

### 주요 시스템
- **라운드 시스템**: 5라운드로 구성된 게임 진행
- **골드 시스템**: 유닛 및 아이템 구매
- **아이템 인벤토리**: 전략적 아이템 관리
- **정보 UI**: 아군/적군 정보 실시간 확인

## 프로젝트 구조

```
Assets/
├── ArtWork/           # 게임 아트 리소스
│   ├── Battle/        # 전투 이미지
│   ├── board라운드/    # 보드 배경 이미지
│   ├── item/          # 아이템 스프라이트 및 프리팹
│   ├── piece/         # 유닛 스프라이트
│   ├── screen/        # UI 화면 이미지
│   └── teamInfo/      # 팀 정보 UI 이미지
├── FishNet/           # 네트워킹 라이브러리
├── Prefabs/           # 게임 오브젝트 프리팹
└── Scripts/           # 게임 로직 스크립트
    ├── board.cs       # 보드 게임 메인 로직
    ├── GameUI.cs      # UI 관리
    ├── Server.cs      # 서버 네트워킹
    └── Client.cs      # 클라이언트 네트워킹
```

## 게임 플레이 방법

1. **게임 시작**
   - Host: 서버를 열고 상대 접속 대기
   - Connect: IP 주소 입력하여 호스트에 연결

2. **게임 진행**
   - 턴당 10회 이동 가능
   - 상점에서 유닛/아이템 구매
   - 전략적 위치 선점 및 전투

3. **승리 조건**
   - 5라운드 종료 시 더 많은 승리 획득
   - 상대 유닛 전멸

## 개발 환경

- **Unity Version**: 2021.3 LTS 이상
- **Platform**: Windows, WebGL
- **Network**: LAN/Internet

## 빌드 및 실행

### 에디터에서 실행
1. Unity Hub에서 프로젝트 열기
2. `Assets/Scenes/SampleScene` 열기
3. Play 버튼 클릭

### 빌드
1. File > Build Settings
2. Platform 선택 (Windows/WebGL)
3. Build 실행
